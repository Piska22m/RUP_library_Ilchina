# Глоссарий проекта
| Термин| Что означает|
|--------|--------------|
| Читатель| Студенты или сотрудники техникума, который берёт книгу |
| Книга | Издание,котрое есть в библиотеке (может быть в нескольких экземплярах) |
| Экземпляр  | Конкретный физический книжный экземпляр |
| Бронь | Временное закрепление книги за читателем (обычно на 2 дня) |
| Выдача | Передача книги читателю на срок |
| Задолжность | Просрочка влвзрата книги |
| Каталог | Список всех книг с возможностью поиска |
